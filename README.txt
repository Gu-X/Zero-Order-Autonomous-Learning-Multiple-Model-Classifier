# Zero-Order-Autonomous-Learning-Multiple-Model-Classifier
This is the python code of the zero-order autonomous learning multi-model classifier (python 3.6). 
This work is described in:
==============================================================================================
P. P. Angelov and X. Gu, “Autonomous learning multi-model classifier of 0-order (ALMMo-0),” in 
IEEE International Conference on Evolving and Autonomous Intelligent Systems, 2017, pp. 1–7.
